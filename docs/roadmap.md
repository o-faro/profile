# Roadmap

## i18n

### Step 1 — UI translations

1. Implement a translation system that reads the locale from the URL (e.g. `/de`) and serves the correct UI strings
2. Provide the actual UI string translations (labels, navigation, buttons, etc.)

### Step 2 — Content translations

Add English equivalents of the German content files:

- `content/projects/de/` — all project description files
- `content/pages/de/` — `home.md`, `home.teaser.md`
